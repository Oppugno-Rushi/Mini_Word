# MiniWord Button Manual

## File Operations
- **New Document** (`file_new`): Create a new document
- **Open Document** (`file_open`): Open an existing document
- **Save Document** (`file_save`): Save current document
- **Print Document** (`file_print`): Print current document
- **Close Document** (`file_close`): Close current document

## Edit Operations
- **Undo** (`edit_undo`): Undo last action
- **Redo** (`edit_redo`): Redo last undone action
- **Cut** (`edit_cut`): Cut selected text
- **Copy** (`edit_copy`): Copy selected text
- **Paste** (`edit_paste`): Paste clipboard content
- **Find** (`find`): Find text in document
- **Replace** (`replace`): Find and replace text
- **Clear Format** (`clear_format`): Remove formatting from selected text

## Format Operations
- **Bold** (`bold`): Make text bold
- **Italic** (`italic`): Make text italic
- **Underline** (`underline`): Underline text
- **Strikethrough** (`strikethrough`): Strikethrough text
- **Superscript** (`superscript`): Make text superscript
- **Subscript** (`subscript`): Make text subscript
- **Text Color** (`text_color`): Change text color
- **Highlight** (`highlight`): Highlight text background
- **Format Painter** (`format_painter`): Copy and apply formatting

## Alignment Operations
- **Align Left** (`align_left`): Align text to left
- **Align Center** (`align_center`): Center align text
- **Align Right** (`align_right`): Align text to right
- **Align Justify** (`align_justify`): Justify text alignment

## Layout Operations
- **Columns** (`layout_columns`): Create text columns
- **Page Break** (`layout_page_break`): Insert page break
- **Page Setup** (`layout_page_setup`): Configure page settings
- **Header & Footer** (`layout_header_footer`): Add headers and footers

## List Operations
- **Bulleted List** (`bulleted_list`): Create bulleted list
- **Numbered List** (`numbered_list`): Create numbered list
- **Increase Indent** (`indent_increase`): Increase paragraph indent
- **Decrease Indent** (`indent_decrease`): Decrease paragraph indent

## Insert Operations
- **Insert Image** (`insert_image`): Insert image into document
- **Insert Link** (`insert_link`): Insert hyperlink
- **Insert Table** (`insert_table`): Insert table
- **Insert Chart** (`insert_chart`): Insert chart
- **Insert Symbol** (`insert_symbol`): Insert special symbol
- **Insert Equation** (`insert_equation`): Insert mathematical equation

## Review Operations
- **New Comment** (`review_comment`): Add comment to document
- **Spelling Check** (`review_spelling`): Check document spelling

## View Operations
- **Zoom In** (`view_zoom_in`): Increase zoom level
- **Zoom Out** (`view_zoom_out`): Decrease zoom level
- **Preview** (`view_preview`): Preview document
- **Help** (`help`): Show help information

## Application Settings
- **Application Settings** (`app_settings`): Configure application settings
