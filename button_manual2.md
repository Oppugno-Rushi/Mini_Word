# MiniWord Button Manual 2 - Advanced Features

## Advanced Formatting
- **Clear All Formatting** (`mw2_clear_all`): Remove all formatting from selected text
- **Format Painter** (`mw2_format_painter`): Copy and apply comprehensive formatting
- **Subscript** (`mw2_subscript`): Make text subscript
- **Superscript** (`mw2_superscript`): Make text superscript

## Document Management
- **Cloud Sync** (`mw2_cloud_sync`): Sync documents to cloud storage
- **Share Document** (`mw2_share`): Share document with others
- **Template Library** (`mw2_template_library`): Access document templates

## Layout & Design
- **Columns** (`mw2_columns`): Create multi-column layouts
- **Page Setup** (`mw2_page_setup`): Configure page settings
- **Header & Footer** (`mw2_header_footer`): Add headers and footers
- **Section Break** (`mw2_section_break`): Insert section breaks
- **Page Break** (`layout_page_break`): Insert page breaks
- **Margins** (`mw2_margin`): Adjust page margins
- **Line Spacing** (`mw2_line_spacing`): Adjust line spacing
- **Paragraph** (`mw2_paragraph`): Format paragraphs

## Table Operations
- **Merge Cells** (`mw2_table_merge`): Merge table cells
- **Split Cells** (`mw2_table_split`): Split table cells

## View & Interface
- **Sidebar Toggle** (`mw2_sidebar_toggle`): Toggle sidebar visibility
- **Ruler Toggle** (`mw2_ruler_toggle`): Toggle ruler visibility
- **Show Invisibles** (`mw2_show_invisibles`): Show invisible characters
- **Focus Mode** (`mw2_focus_mode`): Enable distraction-free writing
- **Night Mode** (`mw2_night_mode`): Toggle dark mode

## Document Analysis
- **Character Count** (`mw2_char_count`): Count characters in document
- **Word Count** (`mw2_word_count`): Count words in document

## Document Protection
- **Protect Document** (`mw2_protect_doc`): Protect document from editing
- **Password Protection** (`mw2_password`): Add password protection

## Track Changes
- **Track Changes On** (`mw2_track_on`): Enable change tracking
- **Track Changes Off** (`mw2_track_off`): Disable change tracking
- **Accept Changes** (`mw2_accept_change`): Accept tracked changes
- **Reject Changes** (`mw2_reject_change`): Reject tracked changes

## Styles & Themes
- **Styles** (`mw2_styles`): Apply document styles
- **Application Settings** (`app_settings`): Configure application settings
